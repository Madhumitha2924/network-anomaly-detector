# network-anomaly-detector
Nokia Bell Labs Relevance
Nokia's Bell Labs AIOps platform monitors 5G network telemetry in real time to detect:

Network anomalies in Radio Access Networks (RAN)
DDoS/intrusion events in core network flows
KPI degradation across thousands of network nodes

This project demonstrates the same core techniques — statistical baselining, rule-based detection,
and IP-level behavioural analysis — applied to network packet data.

Skills Demonstrated

Java OOP: Clean 4-class architecture (Model, Engine, Generator, Reporter)
Statistical ML: Z-score, IQR, percentile computation — implemented from scratch in Java
Data Structures: HashMap, HashSet, Stream API, Comparator, Optional
Algorithms: Real-time streaming analysis, multi-feature scoring, threshold tuning
Domain Knowledge: Network protocols, TCP/IP, DDoS patterns, port scanning, C2 traffic
Software Design: Single-responsibility principle, clear pipeline, testable components
